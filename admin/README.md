# FDM Tracker — Admin Dashboard

Next.js 14 App Router + TanStack Query + Tailwind.

## Dev

```bash
cp .env.example .env.local   # edit NEXT_PUBLIC_API_BASE if the API isn't on :8000
npm install
npm run dev                  # opens on http://localhost:3000
```

Log in as any admin-role user (e.g. `digital@fourdm.com`).

## Build

```bash
npm run typecheck
npm run build
npm start
```
